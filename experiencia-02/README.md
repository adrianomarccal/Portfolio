# 👁️ Experiência 02: Classificação Visual e Análise de Viés em Machine Learning
## 📝 Descrição do Projeto
O objetivo desta atividade foi explorar na prática o treinamento de modelos de Visão Computacional e, mais importante, analisar criticamente as implicações éticas das decisões algorítmicas. Utilizando a plataforma Google Teachable Machine, treinamos um modelo para categorizar imagens em duas classes: "Rebelde" e "Bom aluno", utilizando 10 amostras para cada. O experimento serviu para demonstrar como as IAs podem herdar e amplificar preconceitos humanos.

## 🚀 Tecnologias e Conceitos Utilizados
* **Ferramenta:** Google Teachable Machine
* **Conceitos:** Machine Learning, Visão Computacional, Viés Algorítmico (Algorithmic Bias), Ética em Inteligência Artificial.

## 📊 Resultados e Aprendizados Críticos
Durante os testes, o modelo classificou um indivíduo na categoria "Rebelde" com 96% de confiança, levantando discussões importantes registradas em nosso relatório:
* **Mecanismo do Viés:** Compreendemos que quando alimentamos a IA com poucas imagens e com perfis enviesados, o algoritmo associa erroneamente características físicas (como cor da pele ou vestimenta) a comportamentos. A máquina não avalia a essência, apenas reflete o preconceito dos dados selecionados.
* **Consequência Social:** Refletimos sobre como a redução de indivíduos a rótulos por máquinas cria barreiras reais, como exclusão em vagas de emprego e repetição de ciclos históricos de discriminação.
* **Ação Mitigadora:** Concluímos que a presença humana, a diversidade na equipe de curadoria de dados e a empatia são os verdadeiros filtros éticos da tecnologia. Dados visuais não devem ser usados para julgar caráter.
